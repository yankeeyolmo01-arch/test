# Portfolio Website

This is a portfolio website project that showcases an individual's work, skills, and experiences. The website consists of six main pages, each serving a specific purpose:

## Project Structure

- **src/pages/index.html**: The homepage of the portfolio, featuring an introduction and links to other pages.
- **src/pages/about.html**: A page dedicated to providing background information and experience.
- **src/pages/projects.html**: A showcase of completed projects with descriptions and links.
- **src/pages/blog.html**: A collection of blog posts or articles related to the individual's work or interests.
- **src/pages/contact.html**: A page with contact information and a form for inquiries.
- **src/pages/resume.html**: A display of the individual's resume, highlighting skills, education, and work experience.

## Components

- **src/components/header.html**: The header component for consistent navigation across all pages.
- **src/components/footer.html**: The footer component for consistent footer content across all pages.

## Styles

- **src/styles/main.css**: The CSS file that defines the layout, colors, fonts, and overall design of the website.

## Scripts

- **src/scripts/main.js**: JavaScript code for interactivity and dynamic behavior on the website.

## Data

- **data/projects.json**: Structured data about the projects, used to dynamically populate the projects page.

## Installation

To get started with this project, clone the repository and install the necessary dependencies:

```bash
git clone <repository-url>
cd portfolio-website
npm install
```

## Usage

After installation, you can open the `index.html` file in your browser to view the portfolio website. Use the navigation links to explore the different pages.
